## Overview
World Ranks is an interactive web application that allows users to browse information about all countries in the world. Users can search for a country, sort countries by population or area, and filter by regions. Clicking on a country displays **detailed information** including capital, subregion, population, area, currency, and a Google Maps link.

---

## Responsive Design
The project is fully responsive and works smoothly on all devices:

- **Desktop:** Full table view for all countries.  
- **Tablet:** Flexible column layout for better readability.

## ⚡ Features

- Search countries by name.  
- Sort countries by **Population** or **Area**.  
- Filter countries by **Region** (e.g., Asia, Europe, Africa…).  
- Country details page includes:  
  - Flag  
  - Name  
  - Population  
  - Area  
  - Capital  
  - Subregion  
  - Currency  
  - Google Maps link  
- Fully **responsive** for Desktop, Tablet, and Mobile.  
- Error handling for API issues with user-friendly messages.
